ESP32 Bluetooth Controlled Car

In this project, I built a small smart car using the ESP32 microcontroller. The car can be controlled wirelessly through Bluetooth using a smartphone. I designed the system so that the phone sends commands to the ESP32, and the ESP32 controls the motors to move the car forward, backward, left, and right. This project demonstrates how to use Bluetooth communication, motor drivers, and microcontroller programming to create a simple remote-controlled vehicle. It is a great beginner project for learning robotics, Arduino programming, and wireless control systems.

Main features:

Control the car using a smartphone.

Bluetooth communication with ESP32.

Forward, backward, left, and right movement.

Simple and affordable DIY electronics project.

This project is perfect for anyone interested in electronics, robotics, and DIY technology.

Code for ESP32


#include "BluetoothSerial.h"

BluetoothSerial SerialBT;

int IN1 = 26;
int IN2 = 27;
int IN3 = 14;
int IN4 = 12;


void forward();
void back();
void left();
void right();
void stopCar();

void setup() {
  pinMode(IN1, OUTPUT);
  pinMode(IN2, OUTPUT);
  pinMode(IN3, OUTPUT);
  pinMode(IN4, OUTPUT);

  stopCar();

  SerialBT.begin("ESP32_CAR"); 
}

void loop() {
  if (SerialBT.available()) {
    char c = SerialBT.read();

    if (c == 'F') forward();
    else if (c == 'B') back();
    else if (c == 'L') left();
    else if (c == 'R') right();
    else if (c == 'S') stopCar();
  }
}

void forward(){
  digitalWrite(IN1,1); 
  digitalWrite(IN2,0);
  digitalWrite(IN3,1); 
  digitalWrite(IN4,0);
}

void back(){
  digitalWrite(IN1,0); 
  digitalWrite(IN2,1);
  digitalWrite(IN3,0); 
  digitalWrite(IN4,1);
}

void left(){
  digitalWrite(IN1,0); 
  digitalWrite(IN2,1);
  digitalWrite(IN3,1); 
  digitalWrite(IN4,0);
}

void right(){
  digitalWrite(IN1,1); 
  digitalWrite(IN2,0);
  digitalWrite(IN3,0); 
  digitalWrite(IN4,1);
}

void stopCar(){
  digitalWrite(IN1,0);
  digitalWrite(IN2,0);
  digitalWrite(IN3,0);
  digitalWrite(IN4,0);
}
